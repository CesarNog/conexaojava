Instale a SDK do Android em seu micro e o plugin ADT em seu Eclipse segundo o link:
http://developer.android.com/sdk/installing.html


Procure na barra superior do seu Eclipse o ícone do Android Virtual Device Manager seguindo os passos:
<ul>
	<li>- Escolha um nome</li>
	<li>- No target escolha: Google APIs (Google Inc.) - API Level 7
		(IMPORTANTE PARA TERMOS O SUPORTE A MAPAS!)</li>
	<li>- Escolha o tamanho do SDCard que preferir, bem como a resolução de tela.</li>
<ul>
	
Rode a aplicação com o botao direito -> Run As -> Android Application

Para testar a integração com o Aplicativo Server Side em VRaptor basta descomentar
as linhas na ConexaoJavaActivity. Lembre-se de startar o server!
